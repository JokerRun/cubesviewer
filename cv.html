CubesViewer

Visual data analysis and explore tool

Welcome to CubesViewer Explorer Demo. This is a visual, web-based tool application for exploring and analyzing data. CubesViewer can be used for data exploration and data auditory, generation of reports, chart design and embedding, and as a (simple) company-wide analytics application. This demo shows a reduced set of the full applicaton features.

You can use the menu and interface to navigate this example dataset, which is composed of fictional sales of a web store during years 2012-2013 and also fictional web visit data.

To begin, explore the views, or select a "Example view" or a "Cube" from the list on the left.
