CubesViewer

Data exploration and visualization tool

Demo version (server-side features are not available)

Welcome to CubesViewer Explorer Demo. This is a visual, web-based tool for exploring and analyzing datasets served by the Cubes OLAP Framework. CubesViewer can be used for data exploration and auditory, generation of reports, chart designing and embedding, and as a (simple) company-wide analytics application. This demo shows a reduced set of the full applicaton features.

You can use the menu and interface to navigate this example dataset, which is composed of fictional sales of a web and also fictional web visit data.

Explore the example views, or create a new view from scratch selecting "Cubes" from the top menu. New views are added at the bottom.
